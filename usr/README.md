# نظام إدارة متجر الحواسيب

تطبيق لسطح المكتب (Windows) لإدارة متاجر الحواسيب. يتميز التطبيق بتصميم عصري بالوضع المظلم (Dark Mode)، ويدعم اللغة العربية (من اليمين إلى اليسار).

## المميزات
- واجهة مستخدم عصرية وسريعة الاستجابة.
- شريط جانبي للتنقل (لوحة القيادة، المخزون، العملاء، الإعدادات).
- لوحة قيادة تعرض ملخصاً وإحصائيات المتجر.
- قاعدة بيانات محلية (SQLite) لضمان الخصوصية وعدم الحاجة للاتصال بالإنترنت.
- أزرار دائرية وأيقونات مميزة لكل قسم.
- دعم كامل للغة العربية (RTL).

## التقنيات المستخدمة
- Flutter لسطح المكتب (Windows)
- SQLite (sqflite_common_ffi) لقاعدة البيانات المحلية
- Dart

## الإعداد والتشغيل
1. تأكد من تثبيت Flutter SDK.
2. قم بتشغيل الأمر `flutter pub get` لتثبيت الحزم.
3. لتشغيل التطبيق على نظام التشغيل Windows، استخدم الأمر `flutter run -d windows`.
4. لإنشاء ملف تنفيذي مستقل (EXE)، قم بتشغيل الأمر `flutter build windows`.
